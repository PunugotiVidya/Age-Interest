FIND INTERESTS

In Find Interest Page, the User will be asked with following form details.

  1. Amount
  2. Rate
  3. Start Date
  4. Current Date

The Amount Represent the amount given to the person for Simple Interest

The Rate represent the rate at which amount interest is calculated

TheStart Date represent the date at which the amount was taken

The Current Date represent the End Date for calculation.

After click on Find Interest the Interest will be calculated and will be shown at the top


CALCULATE AGE

In Calculate Age Page, the User will be asked with following form details.

  1. Born DOB
  2. Today Date

TheBorn DOB represent the Date at which user Born

The Current Date represent the Today Date for calculation.

After click on Calculate Age the Age will be calculated and will be shown at the top


Store Interest

In Store Interests, It can able to store the interest and later displayed in list Interests in List Interest navigation bar. It is using the following form fields.

  1. Taken Person
  2. Start Date
  3. Taken Amount
  4. Rate

LIST INTERESTS

In List Interests, It will List all the Interests which are created using Store Interests, User with Delete button for each lender User. If User wants to Delete the specific taken person he can delete by using delete button.

If the User clicks on the name of the lender person, then it will display all the amounts with interest taken by that particular person.


SIGN OUT

If the User clicks on Sign Out page, the User will be logged out and will ask again the User to login again to get the Required Age&Interest Forms.
